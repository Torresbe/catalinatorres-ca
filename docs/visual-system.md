# Visual system — I · Manuscript & Typewriter

Dirección escogida después de rechazar 3 opciones genéricas (Fraunces-type) y revisar 4 alternativas literarias distintivas.

## Tipografía

- **Display headings:** Libre Bodoni (serif con contraste dramático)
- **Body / párrafos:** Courier Prime (typewriter monospace) — **NO sans-serif**
- **Labels técnicos:** Courier Prime

## Tokens de color

```css
--paper: #faf6ec;      /* warm cream */
--ink: #2a2420;        /* body text */
--red: #b8000a;        /* ERROR codes, correction marks, accents */
--marginalia: #7a6a54;
--divider: #c4b89f;    /* dashed borders */
```

## Decoración

- Marcas rojas de corrección editorial
- Labels tipo `MS. folio 1`, `[§3]`
- Em dashes, flechas `→`
- Divisores dashed

## Reglas duras

- **Cero shadows**
- **Cero gradients**
- **Cero rounded corners > 4px**
